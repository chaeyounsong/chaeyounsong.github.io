## 모든 CSS 설정 내역은 공유된다.

예를 들자면

```
<h1>Hello world!</h1>

```
에 직접적으로 selector를 h1으로 잡아주면
다른 js css에도 적용된다.
그래서 class로 따로 운영해준다.
